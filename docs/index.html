Database

A Database Object which create files to store data on the database

new Database(name: string, file_path: string?)
Parameters
name (string) Name of the Database
file_path (string? = __dirname) The path where the data is stored. It's optional and it takes the currect directory by default.
Example
let database1 = new Database("Database name");
let database2 = new Database("Database name", "/something/something")
