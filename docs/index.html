filename (string) The file(shard) in which the data is stored
key (string) The key of the data which will be deleted.
seconds (Number) The no. of seconds after which it will be deleted.

sleepProcessCreate

This function puts a createData process to sleep for 5 seconds.

sleepProcessCreate(obj: Object, key: string, value: Object, seconds: Number): Promise
Parameters
obj (Object) the current database object for which the function has been called.
key (string) the key for the data which will be storted in the database.
value (Object) the JSON value which will be stored in the database.
seconds (Number) if TTL is enabled for the data then it contains the no. of seconds otherwise undefined.
Returns
Promise: returns a promise which will be resolved after the createData function is successfully executed.

Database

A Database Object which create files to store data on the database

new Database(name: string, file_path: string?)
Parameters
name (string) Name of the Database
file_path (string? = __dirname) The path where the data is stored. It's optional and it takes the currect directory by default.
Example
let database1 = new Database("Database name");
let database2 = new Database("Database name", "/something/something")
